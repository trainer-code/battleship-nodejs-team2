Instructions

  1. Download the battleship zip package from the link above
  2. Unzip the package using WinZip/7-Zip/tar or similar software
  3. Open the extracted directory. You need to have a bin directory

You need to install nodejs to run this game

After downloading the battleship game, write tar -zxvf battleship.tar.gz to extract the file to the current directory.

Go to build directory by writing cd build in the terminal

Then run the following command in a terminal to install all the necessary packages: npm install

Then run the following command in a terminal to run the application node ./index.js
